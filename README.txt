🧬 ГЕНОМНЫЕ ВОЙНЫ (Genome Wars)
===================================

Генетическая карточная стратегия для Telegram.
Вы управляете эволюцией и сражаетесь за выживание своего царства!


📋 Описание
-----------
«Геномные Войны» — это пошаговая карточная игра, в которой вы выбираете
одно из четырёх царств живой природы и сражаетесь с противником (ботом).
Ваша цель — уничтожить планету противника, используя уникальные существа
и тактические приёмы.


🎮 Царства
----------
🐾 Animalia (Животные)  — сила и хищничество
🌿 Plantae (Растения)   — защита и регенерация
🍄 Fungi (Грибы)     — паразитизм и контроль
🦠 Bacteria (Бактерии)  — мутации и токсины


⚙️ Механика
-----------
🎯 Цель: Уничтожить планету противника (10 HP).

🧪 Мутагены:
  • В начале каждого раунда вы получаете Мутагены = номер раунда
  • Неиспользованные Мутагены сгорают в конце раунда
  • Базовый лимит Мутагенов — 10

🃏 Карты:
  • В колоде 20 карт (по 2 копии каждого существа царства)
  • В начале игры вы получаете 4 карты
  • Каждый ход вы добираете 1 карту
  • Если колода пуста — планета получает урон от истощения (1, 2, 3...)

🏛️ Поле боя (3 линии):
  • МО (Место Обитания) — 3 слота для размещения существ + одно место занимает планета
  • Экотон — 4 слота для ближнего боя
  • МО противника — 4 слота

⚔️ Бой:
  • Обычные существа атакуют из экотона → МО противника
    или из МО → экотон (оба получают урон)
  • Дистанционная атака: атака из МО → МО без ответного урона
  • Урон = Атака − Броня цели
  • Охрана — защищает соседние существа
  • Атаковать пустые слоты нельзя


🤖 Команды
----------
/start     — Приветствие и информация
/newgame   — Начать новую игру (выбор царства)
/rules     — Правила игры
/help      — Справка
/endgame   — Завершить текущую игру
/genome    — Древо геномов (прогресс)
/clear     — Очистить чат


🛠️ Технический стек
--------------------
• Python 3.11
• Aiogram 3.x — Telegram Bot API
• SQLite — хранение игр
• Yandex Cloud Functions — бессерверный деплой
• SourceCraft — CI/CD


🚀 Деплой
---------
Бот развёрнут как Yandex Cloud Function.
При пуше в ветку master/main автоматически запускается CI/CD,
который собирает и деплоит новую версию функции.
